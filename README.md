# 🤖 Find My Bot

> Complete directory of Telegram and WhatsApp bots. Compare, discover and find the perfect bot for your needs.

🌐 **Site:** [https://heimdall-bot.github.io/find-my-bot](https://heimdall-bot.github.io/find-my-bot)

## 📋 About

Find My Bot is a multilingual directory and comparison site for Telegram and WhatsApp bots. Available in English, Portuguese (BR), and Spanish.

Features:
- 🔍 **150+ bots reviewed** across 10 categories
- ⚡ **Head-to-head comparisons** between similar tools
- 🆓 **Free & open-source highlights**
- 📊 **Unbiased rankings** — no sponsored placements
- 🎯 **Role-based recommendations** — startup, freelancer, e-commerce, agency
- 🌍 **Multilingual** — English, Português (BR), Español
- ⚡ **Boost system** — promote your bot with Spark, Flame, or Blaze plans
- 🔐 **Admin panel** — manage bot listings and boost subscriptions

## 🚀 Tech Stack

- HTML5 semantic
- CSS3 with variables and responsive design
- Vanilla JavaScript (no dependencies)
- Dark modern theme
- i18n multilanguage support (EN, PT-BR, ES)
- LocalStorage admin panel
